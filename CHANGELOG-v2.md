# Companion App v2 — Complete UI Redesign

**Category:** feature

---

The desktop companion app has been completely redesigned with a modern, polished interface:

- **Setup Wizard** — New 3-step guided setup (Connect → Game → Ready) with token validation, auto-detect WoW installation, and a clear progress indicator.

- **Settings** — New sidebar layout with 4 tabs: Status (sync stats, quick actions), Connection (token test, API status), Game (multiple WoW installations, sync interval, file watcher toggle), and General (startup preferences, auto-update, about & links).

- **System Tray** — Custom popup menu replacing the native OS menu. Shows sync status, character avatars with class colors, recent activity, and realm-grouped character list. Click any character to open their build in the browser. Sync Now button with spinning animation and visual feedback.

- **Multi-Install Support** — Add a second WoW installation path. Each installation gets its own file watcher. Characters from all installations are synced together.

- **Debug Log** — Redesigned with pill-style filter buttons, colored type badges (SYNC/INFO/WARN/ERROR), entry count badge, and "Showing X of Y" footer.

- **Quality of Life** — Save settings without closing the window. Discord link in General tab. Auto-resize tray popup. Collapsible realm groups in character list.
